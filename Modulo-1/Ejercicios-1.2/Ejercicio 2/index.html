Lenguajes de programación

HTML es uno de los lenguajes de programación que vamos a aprender durante este curso. Es un lenguaje de marcado, en el que el código se utiliza para estructurar y desplegar una página web y sus contenidos.

Otra cosa que vamos a aprender durante el curso es CSS (Cascading Style Sheets), que es un lenguaje que describe la presentación de los documentos estructurados en hojas de estilo. Describe cómo se va a mostras un documento en la pantalla.

Por último, aprenderemos JavaScript, que es un lenguaje de programación que se utiliza principalmente para crear páginas web dinámicas.

Cada uno de estos lenguajes tiene sus particularidades.

Iremos profundizando en ellos a lo largo del curso.
